# Student Web Project

## Project Description
This is a beginner web development project demonstrating basic Git repository management and web technologies.

## Contents
- **index.html**: Main HTML page with basic structure
- **styles.css**: CSS stylesheet for styling the web page
- **.gitignore**: Configuration file to exclude unnecessary files from version control

## Technologies Used
- HTML5
- CSS3
- Git

## Purpose
This project is created for learning Git basics and repository management, including:
- Initializing a Git repository
- Staging and committing files
- Creating meaningful commit messages
- Managing .gitignore files
- Viewing commit history
